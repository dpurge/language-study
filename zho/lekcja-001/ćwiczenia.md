# Ćwiczenia

## Ćwiczenie 1 — Tłumaczenie zdań (chiński → polski)

Przetłumacz następujące zdania na język polski:

1. 人工智能如今在现代冲突与监控中扮演着重要角色。
2. 该技术能分析卫星图像、识别潜在目标。
3. 此举引发强烈批评，并掀起关于隐私与控制权的讨论。
4. 专家警示严重风险。
5. 全球亟需建立明确的监管框架。

## Ćwiczenie 2 — Uzupełnij luki

Wstaw odpowiednie słowo z listy: `被 / 使 / 并 / 据报道 / 亟需`

1. ______，美国近期在伊朗运用了人工智能。
2. 人工智能在战场之外同样______广泛应用。
3. 该技术能预测敌方反应，______提升了决策速度。
4. 此举引发批评，______掀起了关于隐私的讨论。
5. 全球______建立监管框架。
